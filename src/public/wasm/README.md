# sim_core WASM artefacts

The files `sim_core.js` and `sim_core.wasm` are generated by the CI pipeline
(`.github/workflows/build-wasm.yml`) and committed here automatically.

To build locally, install [Emscripten](https://emscripten.org/) and run:

```bash
cd cpp
make wasm
```

The loader (`src/wasm/index.ts`) probes for `sim_core.wasm` at startup.
If the files are absent, the app continues with JS-only physics.
