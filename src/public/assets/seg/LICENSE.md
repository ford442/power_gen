# SEG housing shell (`housing-shell.glb`)

Procedural geometry generated by `npm run generate:housing-glb` for the hybrid
glTF + procedural pipeline.

- **License:** MIT (same as the power_gen repository)
- **Authoring:** parametric boxes in `scripts/generate-seg-housing-glb.mjs` — replace
  with artist-authored glTF/GLB exported from Blender/Fusion when available.
- **Anchors:** `extras.power_gen.anchors` in the glTF root node (see `docs/GLTF_ASSETS.md`)

No third-party mesh data is bundled beyond this repo-generated asset.
